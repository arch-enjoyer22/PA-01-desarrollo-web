Â¿Que es?

Arch es un sistema operativo de codigo abierto basado en Linux, se caracteriza por poseer un proceso de instalacion totalmente transparente, es decir, que el proceso de instalacion se realiza manualmente (sin interfaz grafica), integrando programas y configurando el sistema a partir de lineas de codigo. Esto resulta en una instalacion limpia sin software innecesario o que no deseemos. Tambien lo vuelve un sistema poco intuitivo y orientado a usuarios con mas conocimientos.
